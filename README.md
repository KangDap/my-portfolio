## My Portfolio 'v')

Umm yeah so why I create this web portfolio is because I'm tired with all of my projects assignment (weird huh).

Basically all of the stuff in here is about myself, like anything I've done in my life. Feel free to look around if you like.

This web use [Next.js](https://nextjs.org) as the framework, with [Resend](https://resend.com/) as the mailer. Most of the components I use in this web were taken from [Animate UI](https://animate-ui.com/) and [Magic UI](https://magicui.design/).

This web also use [GSAP](https://gsap.com/) for animate things and [Lenis](https://www.lenis.dev/) for smooth scroll (idk why I don't use GSAP `scrollSmoother` plugins, but whatever).
